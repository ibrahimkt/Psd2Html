Nemu Restaurant Website

A responsive restaurant website built by converting a Photoshop (PSD) design into a fully functional front-end using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

This project focuses on transforming a high-quality Photoshop design into a modern, responsive, and user-friendly website for **Nemu Restaurant**. It ensures pixel-perfect implementation while maintaining performance and usability across devices.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Adobe Photoshop (for design source)

---

## ✨ Features

- 🎨 Pixel-perfect PSD to HTML conversion  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Fast loading and optimized layout  
- 🍔 Modern restaurant UI/UX  
- 🧭 Smooth navigation and interactive elements  
- 🖼️ Image sections for menu, gallery, and promotions
